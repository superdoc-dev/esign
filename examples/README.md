# SuperDoc eSign Examples

## Quick Start

Each example can be run independently:

### React
```bash
cd react/basic
npm install
npm run dev
```

### Vue
```bash
cd vue/basic
npm install
npm run dev
```

### Vanilla JS
```bash
cd vanilla/basic
npm install
npm run dev
```

## What's Included

- **basic/** - Minimal integration showing all features
- **advanced/** - Full application with custom UI (coming soon)
- **typescript/** - TypeScript example (coming soon)
